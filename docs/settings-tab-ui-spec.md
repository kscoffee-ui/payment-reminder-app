# 設定タブUI仕様

このファイルは、カイシュルの幹事側「設定 / イベント情報」画面のUI見本を説明するための仕様メモです。

## 参照画像

- docs/settings-tab-reference.png

## 目的

設定タブは、URL管理画面ではなく、以下に集中する。

- イベント情報の確認
- 幹事からの案内の確認
- イベント情報の編集
- 参加者URLの再共有

## 表示順

1. カイシュルロゴ
2. ロゴ下の薄い区切り線
3. ページタイトル「設定 / イベント情報」
4. イベント概要カード
5. 幹事からの案内カード
6. イベント情報を編集ボタン
7. 参加者を追加するカード
8. 幹事用ボトムタブバー

## イベント概要カード

2×2グリッドで表示する。

- イベント名
- 日付
- 会費
- 支払い

表示例：

- イベント名: 飲み会
- 日付: 2026-05-27
- 会費: ¥4,245
- 支払い: 現金回収

## 幹事からの案内

独立した白カードで表示する。

表示例：

幹事からの案内  
幹事に現金で直接渡してください

任意メモが空、未設定、または「-」の場合は表示しない。

## 編集ボタン

「イベント情報を編集」は、青ベタではなくアウトラインボタンにする。

- 白背景
- 青枠
- 青文字
- 角丸
- 押しやすい高さ

## 参加者追加カード

見出し：

参加者を追加する

説明文：

LINEグループに送ると、参加者が自分で名前を入力して参加できます。

ボタン：

- LINEで共有
- その他のアプリで共有

## 表示しないもの

以下は設定タブに表示しない。

- joinUrl の全文
- adminUrl の全文
- 幹事用管理ページカード
- ステータス説明カード
- 青い「イベント情報」帯
- 任意メモ: -
- URL表示用ボックス

## UI方針

- モバイルファースト
- 白背景
- 角丸カード
- 薄い枠線
- 弱い影
- 余白は広めだが、文字とカードは大きくしすぎない
- 下部タブバーに内容が隠れないようにする
