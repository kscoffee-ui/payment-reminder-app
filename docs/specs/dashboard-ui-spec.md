## この画面の目的

幹事がイベント全体の未払い状況を一目で把握し、必要に応じて未払い者へLINEで催促できるようにする。

この画面では、以下を最優先する。

- 未払い人数を最も目立たせる
- 回収進捗を一目で把握できる
- 未払い者をすぐ確認できる
- LINE催促導線を分かりやすく表示する
- 幹事の心理的負担を減らす

3. 支払い状況サマリーカード

見本画像の「支払い状況サマリー」に近づける。

表示内容:

使用データ:

members.length
unpaidCount
reportedCount
confirmedCount
completionRate

計算ルール:

totalCount = members.length
unpaidCount = status === "unpaid" の人数
reportedCount = status === "reported" の人数
confirmedCount = status === "confirmed" の人数
completionRate = totalCount === 0 ? 0 : Math.round((confirmedCount / totalCount) * 100)

注意:

未払い人数を最も目立たせる
完了率は確認済み人数を基準にする
0人の場合も表示崩れしないようにする
